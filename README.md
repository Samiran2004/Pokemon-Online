## Random Pokemon Details Web App

This web app is built using HTML, CSS, and JavaScript. It fetches data from an external API that provides Pokemon details and displays the relevant details such as name, type, attack power, speed, and HP. The app also includes a button that allows users to generate random Pokemon details.

### Features

- Displays random Pokemon details
- Fetches data from an external API
- Built using HTML, CSS, and JavaScript


### Usage

1. Click the "Generate" button to display random Pokemon details
